Wgui

A tool for new Linux Users.

Download files from the web with wget, but with GUI!

To download, paste this into your terminal: curl matei1413.github.io/wgui-site/getjar.sh | bash

Works with any Linux distro (with wget & java installed)

Where does it download files?

If you double click, it saves files in ~/ but if you run it in terminal it saves to where you started it.

The installer downloads to Documents. If you do not have a Documents folder, click here.
